## CSS Flexbox

### COSA È FLEXBOX
- **[Flexbox](./definizioni/semantica.md#flexbox "Definizione di Flexbox")** (Flexible Box Layout) è un modulo di layout CSS
- Strumento potente e versatile per organizzare elementi in un contenitore
- Facilita la creazione di [layout flessibili e responsivi](./definizioni/semantica.md#layout-responsivo "Definizione di Layout Responsivo")
- Risolve molti problemi comuni di allineamento e distribuzione dello spazio

### CREARE UN CONTENITORE FLEX
Per iniziare con [Flexbox](./definizioni/semantica.md#flexbox "Definizione di Flexbox"), è necessario definire un contenitore flex:

Struttura HTML:
```html
<div class="flexbox-container">
  <div class="flexbox-item flexbox-item-1">Item 1</div>
  <div class="flexbox-item flexbox-item-2">Item 2</div>
  <div class="flexbox-item flexbox-item-3">Item 3</div>
</div>
```

[CSS](./definizioni/semantica.md#css "Definizione di CSS") per attivare Flexbox:
```css
.flexbox-container {
  display: flex;
}
```

**Risultato**: Tutti gli elementi figli si dispongono automaticamente su una singola riga, assumendo la stessa altezza secondo il [Box Model](./definizioni/semantica.md#box-model "Definizione di Box Model").

### CONCETTI FONDAMENTALI: MAIN AXIS E CROSS AXIS

#### Main Axis (Asse Principale)
- Asse principale lungo il quale gli elementi flex sono disposti
- Direzione determinata dalla proprietà `flex-direction`

#### Cross Axis (Asse Incrociato)  
- Asse perpendicolare al main axis
- Utilizzato per l'allineamento secondario degli elementi

Per maggiori dettagli sulla [terminologia Flexbox](./definizioni/semantica.md#terminologia-flexbox "Terminologia Flexbox").

#### Proprietà flex-direction
Controlla la direzione del main axis:
- `row` (default): asse principale orizzontale (da sinistra a destra)
- `column`: asse principale verticale (dall'alto al basso)
- `row-reverse`: asse principale orizzontale invertito
- `column-reverse`: asse principale verticale invertito

```css
.flexbox-container {
  display: flex;
  flex-direction: row; /* o column, row-reverse, column-reverse */
}
```

### ALLINEAMENTO DEGLI ELEMENTI

#### Proprietà justify-content
Allinea gli elementi lungo il **main axis**:

- `flex-start`: allinea all'inizio del contenitore
- `flex-end`: allinea alla fine del contenitore
- `center`: allinea al centro
- `space-between`: distribuisce con spazio equo tra gli elementi
- `space-around`: aggiunge spazio equo attorno a ogni elemento
- `space-evenly`: distribuisce con spazio uniforme tra e attorno agli elementi

```css
.flexbox-container {
  display: flex;
  justify-content: center; /* centra gli elementi */
}
```

#### Proprietà align-items
Allinea gli elementi lungo il **cross axis**:

- `flex-start`: allinea all'inizio del cross axis
- `flex-end`: allinea alla fine del cross axis
- `center`: allinea al centro del cross axis
- `stretch` (default): allunga per riempire l'altezza del contenitore secondo il [Box Model](./definizioni/semantica.md#box-model "Box Model CSS")
- `baseline`: allinea lungo la baseline del testo

```css
.flexbox-container {
  display: flex;
  align-items: center; /* centra verticalmente */
}
```

### PROPRIETÀ DEGLI ELEMENTI FLEX

#### flex-grow
- Determina quanto un elemento deve crescere per riempire lo spazio disponibile
- Valore numerico (default: 0)
- Rapporto proporzionale tra gli elementi

```css
.flexbox-item-1 {
  flex-grow: 2; /* cresce 2 volte più degli altri */
}
.flexbox-item-2 {
  flex-grow: 1; /* crescita normale */
}
```

#### flex-shrink
- Controlla come un elemento si riduce quando lo spazio è limitato
- Valore numerico (default: 1)
- 0 impedisce la riduzione dell'elemento

```css
.flexbox-item {
  flex-shrink: 0; /* impedisce la riduzione */
}
```

#### flex-basis
- Imposta la dimensione iniziale dell'elemento prima della distribuzione dello spazio
- Può essere una lunghezza (px, %, em) o `auto`
- Si basa sui principi del [Box Model](./definizioni/semantica.md#box-model "Box Model CSS")

```css
.flexbox-item {
  flex-basis: 200px; /* dimensione di base 200px */
}
```

#### Proprietà shorthand flex
Combina flex-grow, flex-shrink e flex-basis:

```css
.flexbox-item {
  flex: 1 0 200px; /* grow: 1, shrink: 0, basis: 200px */
}
```

### ALTRE PROPRIETÀ UTILI

#### flex-wrap
Permette agli elementi di andare su più righe quando necessario:
- `nowrap` (default): tutti gli elementi su una riga
- `wrap`: gli elementi vanno a capo se necessario
- `wrap-reverse`: come wrap, ma in ordine inverso

```css
.flexbox-container {
  display: flex;
  flex-wrap: wrap;
}
```

#### order
Cambia l'ordine di visualizzazione senza modificare l'[HTML](./definizioni/semantica.md#html "Definizione di HTML"):
- Valore numerico (default: 0)
- Ordine crescente

```css
.flexbox-item-1 {
  order: 3; /* appare per ultimo */
}
.flexbox-item-3 {
  order: 1; /* appare per primo */
}
```

#### align-self
Sovrascrive `align-items` per un singolo elemento:
- Stessi valori di `align-items`
- Si applica solo all'elemento specifico

```css
.flexbox-item-2 {
  align-self: flex-end; /* si allinea alla fine del cross axis */
}
```

### ESEMPIO PRATICO COMPLETO

```html
<div class="container">
  <div class="item item-1">Header</div>
  <div class="item item-2">Content</div>
  <div class="item item-3">Sidebar</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: stretch;
  height: 500px;
}

.item {
  padding: 20px;
  background-color: #f0f0f0;
  border: 1px solid #ccc;
}

.item-1 {
  flex: 0 0 200px; /* larghezza fissa 200px */
}

.item-2 {
  flex: 1; /* occupa tutto lo spazio rimanente */
}

.item-3 {
  flex: 0 0 150px; /* larghezza fissa 150px */
}
```

Questo esempio mostra come creare un [layout responsivo](./definizioni/semantica.md#layout-responsivo "Layout Responsivo") tipico con header, contenuto principale e sidebar che si adatta al [viewport](./definizioni/semantica.md#viewport "Definizione di Viewport").

### VANTAGGI DI FLEXBOX
1. **Semplicità**: Layout complessi con poche righe di codice
2. **Flessibilità**: Adattamento automatico a diverse dimensioni dello schermo
3. **Controllo preciso**: Allineamento e distribuzione dello spazio ottimali
4. **Responsive**: Ideale per [design responsive](./definizioni/semantica.md#layout-responsivo "Design Responsivo") senza [media queries](./definizioni/semantica.md#media-queries "Media Queries") complesse
5. **Supporto browser**: Ampiamente supportato dai browser moderni

### CONCLUSIONI
[Flexbox](./definizioni/semantica.md#flexbox "Flexbox Layout") è uno strumento fondamentale per il [CSS](./definizioni/semantica.md#css "CSS") moderno che rivoluziona la creazione di layout. Padroneggiando le proprietà come `justify-content`, `align-items`, `flex-grow` e le altre, è possibile creare interfacce flessibili e responsive con facilità.
